# vue-admin-bootstrap

> 基于 Vue jQuery Bootstrap的组件框架

## Build Setup

```bash
# install dependencies
npm install

# 开发环境
npm run dev

# 生成简单的app
npm run build

# 生成文档
npm run build-docs

# 生成合并文件
npm run release

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
